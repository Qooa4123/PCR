# 機器學習作業一：股票報酬率預測 (Machine Learning HW1)

## 1. 專案概述
[cite_start]本專案採用季末截面資料預測下一季的對數報酬率 ($log$ returns) [cite: 2][cite_start]。為了觀察不同季度下的結果差異，模型採用**滾動執行 (Rolling Execution)** 的方式進行回測 [cite: 4]。

---

## 2. 資料與特徵工程

* [cite_start]**資料範圍**：選取 S&P 500 公司於 2024/01/01 至 2025/12/31 之數據 [cite: 9]。
* [cite_start]**樣本數量**：經過資料清理與缺失值處理後，剩餘約 370 間公司 [cite: 10]。
* **特徵因子 (包含 20 餘項)**：
    * [cite_start]**動能類**：1週反轉、1個月反轉、3/6/12個月動能 [cite: 5]。
    * [cite_start]**波動類**：20日/60日波動率、下行波動率 [cite: 5]。
    * [cite_start]**流動性與技術類**：換手率、Amihud 非流動性指標、成交量 Z 分數、MACD、RSI、布林通道等 [cite: 6]。
    * [cite_start]**基本面類**：ROE、營收成長率、資產成長率、應計項目 [cite: 6, 7]。
* **偏誤處理**：
    * [cite_start]以該季中位數填補缺失值，並對 1%~99% 資料做縮尾處理 [cite: 9]。
    * [cite_start]財報資料取至少 90 天前的最近一次數據，以避免**前瞻偏差 (Look-ahead bias)** [cite: 7]。

## 4. 結果探討與討論

* **基本面因子的預測力**：
    * [cite_start]2024Q4 以前因財報缺失，預測性較弱 [cite: 14]。
    * [cite_start]2025Q2 開始 RMSE 明確下滑，代表營收與資產成長率等基本面因子開始提供有效資訊 [cite: 16]。
* **超參數穩定性**：
    * [cite_start]**$k_{CV}$** 波動大，易受當季噪音干擾 [cite: 17]。
    * **$k_{AIC}$** 與 **$k_{Onatski}$** 選擇的主成分數接近因子總數，代表因子間相關性低，存在過擬合 (Overfitting) 風險 [cite: 18]。
* **模型對比**：
    * **PLS1** 的表現多數時間優於 PCR，展現更佳的泛化能力 [cite: 20]。
    * [cite_start]這暗示 20 個因子可能可被歸納為單一的潛在因子 (Latent factor) 共同影響市場 [cite: 20]。
